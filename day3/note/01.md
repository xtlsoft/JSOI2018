## 2. 搜索状态

决定了我们的搜索的对象。影响搜索的效率。思维复杂度和实现复杂度。

```cpp
void dfs (int dep)
{
   if  (到达目标状态)  输出解或者做计数、评价处理;
   else  for (int i=1;  i <= 状态的拓展可能数; ++i)
   {
         if (第i种状态拓展可行)  
         {
              保存现场;
              dfs(dep+1);
              // 恢复现场(回溯);
         }
    }
}
```

## 奶酪

（染色问题）

初始状态： -r <= z <= r

目标状态：h-r <= z <= h+r

状态转移：0 <= dist(P1, P2) <= 2r （dist 为立体坐标系中求点距离公式）
